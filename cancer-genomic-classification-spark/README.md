# Cancer Genomic Classification Framework

## Overview
An advanced distributed machine learning framework for cancer
classification using synthetic genomic expression data. The pipeline
leverages Apache Spark and ensemble learning to evaluate multiple
tree-based classifiers at scale.

## Key Highlights
- Synthetic genomic data simulation with biological patterns
- Distributed training using PySpark MLlib
- Ensemble models: Decision Tree, Random Forest, Gradient Boosting
- Cross-validation and hyperparameter tuning
- ROC-AUC, Accuracy, and F1-score evaluation
- Feature importance analysis and visualization

## Tech Stack
- Python
- Apache Spark (PySpark)
- MLlib
- Pandas, NumPy
- Matplotlib, Seaborn, Plotly

## Models Used
- Decision Tree Classifier
- Random Forest Classifier
- Gradient Boosted Trees

## Evaluation Metrics
- ROC-AUC
- Classification Accuracy
- F1 Score
