# Energy Guardian: Smart Home Predictive Maintenance

## Overview
A machine learning project that simulates smart home energy usage data
and applies feature engineering, anomaly detection, and predictive
models for energy optimization and appliance health monitoring.

## Key Features
- Synthetic IoT energy data generation
- Appliance-level energy analytics
- Feature engineering & anomaly detection (Z-score)
- ML-ready pipeline for predictive maintenance

## Tech Stack
Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn

## Files
- Smart_Home_Energy_Prediction.ipynb
- smart_home_energy_data.csv
