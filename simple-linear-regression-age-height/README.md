# Simple Linear Regression: Age vs Height

## Overview
A beginner-friendly machine learning project that implements
Simple Linear Regression from scratch to analyze and predict
human height based on age.

## Key Steps
- Dataset loading and exploration
- Data cleaning and correlation analysis
- Manual implementation of linear regression
- Parameter learning (alpha & beta)
- Visualization of trained vs untrained models
- Height prediction for new age inputs

## Mathematical Model
y = α + βx + ε  
ŷ = α + βx

## Tech Stack
- Python
- Pandas
- Matplotlib
- NumPy

## Files
- Age_Height_Linear_Regression.ipynb
